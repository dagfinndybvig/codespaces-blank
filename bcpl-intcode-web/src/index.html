About INTCODE

INTCODE is the bytecode format used by the BCPL system. BCPL source code is compiled into INTCODE, which is then executed by a virtual machine interpreter. This web-based VM runs the same INTCODE that would run on the original BCPL systems from the 1970s.

How to Use:

  1. Compile BCPL source to INTCODE (using syni, trni, cgi from bcpl-c64-code-r8)
  2. Load the resulting INTCODE file using the file chooser above
  3. Click "Run Program" to execute
  4. View output in the panel on the right
