About BCPL

BCPL (Basic Combined Programming Language) is a procedural, imperative, and structured programming language designed by Martin Richards in 1966. It was a precursor to C and influenced many modern languages.

Quick Reference:

  • LET START() BE $( ... $) - Main program entry point
  • WRITES("text") - Write a string
  • WRITEF("format", args) - Formatted output
  • *N - Newline, *T - Tab, *S - Space
  • FOR i = 1 TO 10 DO - Loop construct
  • GET "LIBHDR" - Include library header
